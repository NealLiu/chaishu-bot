# 拆书工作流 · 实现计划

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** 构建自动化拆书工作流：AI 搜索/拆解书籍 → 生成结构化 Markdown → 渲染小红书卡片 PNG → 存入 Obsidian 知识库，带自生长推荐引擎。

**Architecture:** n8n 编排工作流 + DeepSeek API 拆书 + MD2Card 渲染图片 + Obsidian Local REST API 存储。推荐引擎通过维护偏好图谱 YAML 实现自生长。

**Tech Stack:** n8n (Docker), DeepSeek API, MD2Card (headless browser 自动化), Obsidian Local REST API, Python/Node.js 辅助脚本

## 全局约束

- 所有文件使用中文内容（书籍名称、标签等）
- Obsidian vault 路径：`/Users/a123/Documents/Obsidian Vault/拆书/`
- Markdown 文件需包含 YAML frontmatter（title, author, tags, date）
- 卡片图片格式：PNG, 1242×1660px
- 推荐引擎每次拆书后更新偏好图谱

你是一位专业的书籍拆解专家。你的任务是将一本书提炼为结构化的知识卡片内容。

    "具体的可执行行动1",
    "具体的可执行行动2",
    "具体的可执行行动3"

  "next_suggestions": [
    {
      "direction": "横向拓展",
      "book": "推荐书名",
      "reason": "一句话关联理由"
    },
    {
      "direction": "深度延伸",
      "book": "推荐书名",
      "reason": "一句话关联理由"
    },
    {
      "direction": "实用落地",
      "book": "推荐书名",
      "reason": "一句话关联理由"
    }
  ]
}

## 拆解要求

1. **深度 vs 可读性**：每个知识点用通俗语言解释核心概念，不要学术化
2. **洞见优先**：每个知识点必须包含一个"原来如此"的洞见时刻
3. **行动导向**：每个知识点配一个可操作的练习，读完就能做
4. **关联推荐**：推荐3本关联书（不同方向），形成知识网络
5. **小红书风格**：标题要有冲击力，内容要紧凑，适合移动阅读

## 风格要求
- 温暖、接地气、不装
- 像朋友在跟你分享一本好书
- 少用术语，多用比喻

```javascript
// scripts/md2card-render.js
const puppeteer = require('puppeteer');
const fs = require('fs');
const path = require('path');

const MD2CARD_EDITOR = 'https://md2card.com/zh/editor';
const CARD_WIDTH = 1242;
const CARD_HEIGHT = 1660;

async function renderMarkdownToCards(markdownPath, outputDir) {
  const markdown = fs.readFileSync(markdownPath, 'utf-8');
  const baseName = path.basename(markdownPath, '.md');
  
  const browser = await puppeteer.launch({ headless: 'new' });
  const page = await browser.newPage();
  
  // Set viewport to match card dimensions
  await page.setViewport({ width: CARD_WIDTH, height: CARD_HEIGHT });
  
  // Navigate to MD2Card editor
  await page.goto(MD2CARD_EDITOR, { waitUntil: 'networkidle2' });
  
  // Clear default content and paste markdown
  // MD2Card uses CodeMirror editor - find the textarea
  await page.waitForSelector('.CodeMirror');
  await page.evaluate((md) => {
    // MD2Card stores content in CodeMirror instance
    const editor = document.querySelector('.CodeMirror').CodeMirror;
    editor.setValue(md);
  }, markdown);
  
  // Wait for preview to render
  await page.waitForTimeout(2000);
  
  // Click export button, select PNG
  // (selector depends on MD2Card UI, may need adjustment)
  await page.click('[data-action="export-png"]');
  await page.waitForTimeout(3000);
  
  // Handle download - save to outputDir
  // Puppeteer download handling
  const client = await page.target().createCDPSession();
  await client.send('Page.setDownloadBehavior', {
    behavior: 'allow',
    downloadPath: outputDir
  });
  
  // Rename downloaded file
  const downloadedFiles = fs.readdirSync(outputDir)
    .filter(f => f.endsWith('.png') || f.endsWith('.zip'));
  
  for (const file of downloadedFiles) {
    const ext = path.extname(file);
    fs.renameSync(
      path.join(outputDir, file),
      path.join(outputDir, `${baseName} 卡片${ext}`)
    );
  }
  
  await browser.close();
  console.log(`✅ Cards rendered: ${outputDir}`);
  
  return path.join(outputDir, `${baseName} 卡片.png`);
}

// CLI usage
const [,, markdownPath, outputDir] = process.argv;
if (!markdownPath || !outputDir) {
  console.error('Usage: node md2card-render.js <markdown.md> <output-dir>');
  process.exit(1);
}
renderMarkdownToCards(markdownPath, outputDir);
```

### Task 4: 推荐引擎实现

**Files:**
- Create: `scripts/recommendation-engine.js`
- Modify: `books/书籍索引.md`（添加初始偏好图谱）

**Interfaces:**
- Consumes: `books/书籍索引.md` 中的 YAML 偏好图谱
- Produces: JSON `{ book_title, author, reason, strategy }` 供 n8n 或拆书流程使用

- [ ] **Step 1: 编写推荐引擎脚本**

````javascript
// scripts/recommendation-engine.js
const fs = require('fs');
const path = require('path');
const yaml = require('yaml');

const INDEX_PATH = path.join(__dirname, '..', 'books', '书籍索引.md');

function parsePreferences() {
  const content = fs.readFileSync(INDEX_PATH, 'utf-8');
  // Extract YAML block between ```yaml and ```
  const match = content.match(/```yaml\n([\s\S]*?)```/);
  if (!match) throw new Error('未找到偏好图谱 YAML 块');
  return yaml.parse(match[1]);
}

function updatePreferences(bookInfo) {
  const prefs = parsePreferences();
  
  // Merge new book's tags into topic weights
  const newTags = bookInfo.tags || [];
  for (const tag of newTags) {
    const existing = prefs.preferences.topics.find(t => t.name === tag);
    if (existing) {
      existing.weight = Math.min(existing.weight + 0.3, 3.0);
    } else {
      prefs.preferences.topics.push({ name: tag, weight: 1.0, depth: '入门' });
    }
  }
  
  // Update recent focus
  prefs.preferences.recent_focus = newTags.slice(0, 3).join('、');
  
  // Remove filled gaps if book addresses any
  const gapMatches = prefs.preferences.knowledge_gaps.filter(gap =>
    newTags.some(tag => tag.includes(gap) || gap.includes(tag))
  );
  prefs.preferences.knowledge_gaps = prefs.preferences.knowledge_gaps
    .filter(g => !gapMatches.includes(g));
  
  // Add next suggestions as potential new gaps if not already present
  if (bookInfo.next_suggestions) {
    for (const sug of bookInfo.next_suggestions) {
      const sugCategory = sug.reason.split('·')[0] || '';
      if (sugCategory && !prefs.preferences.knowledge_gaps.includes(sugCategory)) {
        prefs.preferences.knowledge_gaps.push(sugCategory);
      }
    }
  }
  
  // Update depth level based on number of books read
  const bookCount = extractBookCount();
  prefs.preferences.depth_level = bookCount <= 3 ? '入门' : 
    bookCount <= 10 ? '进阶' : '深入';
  
  // Write back to index file
  const content = fs.readFileSync(INDEX_PATH, 'utf-8');
  const newYaml = yaml.stringify(prefs);
  const newContent = content.replace(
    /```yaml\n[\s\S]*?```/,
    '```yaml\n' + newYaml + '```'
  );
  fs.writeFileSync(INDEX_PATH, newContent);
  
  return prefs;
}

function extractBookCount() {
  const content = fs.readFileSync(INDEX_PATH, 'utf-8');
  const matches = content.match(/\| \d{4}-\d{2}-\d{2} \|/g);
  return matches ? matches.length : 0;
}

// CLI
const command = process.argv[2];
const bookInfoJson = process.argv[3];

if (command === 'read') {
  console.log(JSON.stringify(parsePreferences(), null, 2));
} else if (command === 'update') {
  const bookInfo = JSON.parse(bookInfoJson);
  const updated = updatePreferences(bookInfo);
  console.log(JSON.stringify(updated, null, 2));
} else {
  console.log('Usage:');
  console.log('  node recommendation-engine.js read');
  console.log('  node recommendation-engine.js update \'{"tags":["心理学"],"next_suggestions":[...]}\'');
}
````
